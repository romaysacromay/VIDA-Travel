# VIDA Travel Brand Guidelines

## Colors

### Primary Colors
- **Primary Blue**: `#0066CC` - Main brand color for headers, CTAs, and primary elements
- **Secondary Blue**: `#00A8E8` - Secondary brand color for accents and gradients
- **Accent Orange**: `#FF6B35` - Call-to-action buttons and highlights

### Neutral Colors
- **Dark Text**: `#1A1A1A` - Primary text color
- **Light Background**: `#F5F5F5` - Section backgrounds
- **White**: `#FFFFFF` - Main background and card backgrounds

## Typography

### Primary Font
- **Font Family**: 'Helvetica Neue', Arial, sans-serif
- **Usage**: Body text, navigation, forms

### Secondary Font
- **Font Family**: 'Georgia', serif
- **Usage**: Headlines, hero text, emphasis

### Font Sizes
- Hero Headline: 3rem (48px)
- Section Headlines: 2.5rem (40px)
- Body Text: 1rem (16px)
- Small Text: 0.875rem (14px)

## Logo

### Logo File
- **File Name**: `vida-logo.png`
- **Location**: `/public/images/vida-logo.png`
- **Recommended Dimensions**: 200px width (height proportional)
- **Format**: PNG with transparency

### Logo Usage
- Use logo in header navigation
- Maintain aspect ratio
- Minimum size: 50px height
- Maximum size: 200px width

## Imagery

### Image Guidelines
- Use high-quality, travel-focused imagery
- Maintain consistent color tone (warm, inviting)
- Optimize images for web (WebP format preferred)
- Use alt text for accessibility

### Image Placements
- Hero section background images
- Destination showcase images
- Testimonial photos (if applicable)

## Spacing

### Container Width
- Maximum content width: 1200px
- Padding: 20px on mobile, 40px on desktop

### Section Spacing
- Section padding: 4rem vertical (64px)
- Element spacing: 1.5rem (24px) between form elements

## Components

### Buttons
- Primary CTA: Accent Orange background, white text
- Secondary: Primary Blue background, white text
- Border radius: 5px
- Padding: 1rem 2rem

### Forms
- Input border: 2px solid #ddd
- Focus border: 2px solid Primary Blue
- Border radius: 5px
- Padding: 0.75rem

## Responsive Design

### Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Mobile Considerations
- Stack navigation vertically
- Reduce hero headline size
- Full-width form inputs
- Touch-friendly button sizes (min 44px)
